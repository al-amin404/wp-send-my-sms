### How to install
Download plugin zip file -> place in plugin folder -> Activate the plugin from WordPress plugins list.

### How to use
Look for **Send SMS** in the admin menu and Navigate to **"Send SMS -> Settings"**. You will see API Username & API Key fields.

Go to your **Send My SMS** dashboard and navigate to [API Credentials](https://sendmysms.net/api-credentials.php) page. Copy the username and key, and paste them on respective field on the plugin's settings page. And, save the settings.

## Woocommerce Order Notification SMS
To automatically send order sms notification for woocommerce orders, navigate to the **"Send SMS -> Settings"** page. There from the list enable sms notification or customize the templates for any order statuses you want. After configuring the templates, hit **Save**. The plugin will send a sms to the customer's billing phone whenever the order status chages to that status.

You can also send custom order sms to customer of any order manually anytime from that page.
